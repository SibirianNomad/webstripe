Поддомены не настроены, поэтому город не меняйте.

1) В админке в разделе "Товары" при редактировании товара (например http://localsite/admin/products/108/edit ) во вкладке "Описания" есть 3 текстовых поля: "Короткое описание", "Описание" и "Характеристики"

Необходимо на страницу http://localsite/admin/products , рядом с поиском добавить 2 кнопки: "Без описания" и "Без характеристик".
- "Без описания" - должна выводить список товаров, у которых пустое поле "Короткое описания" либо "Описание"
- "Без характеристик" - должно выводить список товаров, у которых пустое поле "Характеристики"

2) В админке в разделе "Города" при редактировании города ( http://localsite/admin/cities/24/edit  ) во вкладке "Контакты" есть текстовое поле "Код карты".  Карта выводится внизу главной страницы

Сейчас в это поле вставляется html код для вставки на сайт. Необходимо заменить это на 3 поля: "долгота", "широта" и "метка". При выводе на главную страницу необходимо генерировать html код яндекс карты и вставлять в него координаты из БД. Стиль карты произвольный. Над меткой выводить название из БД (третье поле "метка")

3) Убрать из админки живосайт

4) На главной странице есть блок "КАТАЛОГ ЭЛЕКТРОННЫХ ОШЕЙНИКОВ" и список товаров. Нужно из списка товаров сделать адаптивную карусель с горизонтальной прокруткой. 
Здесь же, в карточке товара, при нажатии "подробнее" перекидывает на страницу товара. Переделать так, чтоб при нажатии на "подробно" , под кнопкой плавно выдвигался блок с Кратким описанием (информация берется из поля "Короткое описание" из п.1. тестового задания)
